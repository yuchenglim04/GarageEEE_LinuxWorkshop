##Command mode
##navigation
```
h          # <
j          # down
k          #^
l          #>
0            # move to beginning of line
$            # move to end of line
W            # move forward one space-delimited word 
w            # move forward one word
B            # move backward one space-delimited word
b            # move backward one word
Backspace    # move backward one character
```
